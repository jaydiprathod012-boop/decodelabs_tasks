## 🚀 Projects Overview

### Project 1: Data Cleaning & Preparation
**Goal:** Clean a raw dataset by handling missing values, duplicates, and incorrect data formats.

**Key Requirements:**
- Identify and handle missing/null values (impute using mean/median/mode — don't just delete)
- Remove duplicate records
- Correct data formats (dates → YYYY-MM-DD, proper case text, numeric precision)
- Document all changes in a Change Log

**Tools:** Python (Pandas) or Excel (Power Query)

---

### Project 2: Exploratory Data Analysis (EDA)
**Goal:** Analyze a dataset to uncover patterns, trends, and distributions.

**Key Requirements:**
- Calculate basic statistics (mean, median, count, std)
- Identify trends and outliers (using IQR or Z-Score method)
- Correlation analysis between variables
- Summarize key observations with business context

**Tools:** Python (Pandas, Matplotlib, Seaborn) or Excel
